Extra flags for 4chan

Originally made for /int/, but we support /sp/, /pol/, and /bant/ now as well. Made by /ref/ and flags contributed by various /ref/ and /int/ posters. Your flag not here? Create an issue on the github page

HOW TO INSTALL (Desktop - Chrome/Firefox/Opera)

  1. Download TamperMonkey for Firefox, TamperMonkey for Chrome or ViolentMonkey for Opera.
  2. Install the Extra Flags script: Extra Flags for int.user.js
  3. Set your region
  4. Reload the page
  5. Post from Quick Reply box (QR) (top form does not work)

Video install guides

If webms do not play in browser: download as .zip.

HOW TO INSTALL (Android Waterfox v55)

Mobile addon for Android Firefox v57 currently needs updating and doesn't work.

  1. Download the Extension on mobile or scan QR below.
  2. Navigate to "about:config" and search for "xpinstall" and flip "xpinstall.signatures.required" to false. After you successfully installed you can flip it back to true.
  3. You MUST open this file from /sdcard to install it, it won't install otherwise. Check webm below!
  4. Set you're region in Options > Tools > Add-ons > Extra Flags for int FFMA > Region
  5. Refresh any 4chan tab you have open.

If webms do not play in browser: download as .zip.
